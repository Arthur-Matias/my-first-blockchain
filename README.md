# My "Hello World" on the Blockchain world


This project is a template of the backbones of a Blockchain

Contains basic logic of Blocks, Agents and transactions

made with TypeScript

## How to run

to run this project you will need typescript and nodejs (you can get help on how to install it [here](https://www.typescriptlang.org/) for TypeScript and [here](https://nodejs.org/en/) for nodejs )

on the project folder run

```bash
    tsc
```

then

```bash
    node bin/index.js
```
